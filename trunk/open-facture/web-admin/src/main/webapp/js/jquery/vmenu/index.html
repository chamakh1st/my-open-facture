vMenu

Vertical submenu

  • Category 1
    • Subitem 1
    • Subitem 2
    • Subitem 3
      • Subitem 3-1
      • Subitem 3-2
      • Subitem 3-3
    • Subitem 4
    • Subitem 5
  • Category 2
    • Subitem 1
    • Subitem 2
Please choose category

MarkUp:

<ul id="vmenu">
    <li><a href="#">Category 1</a>
        <ul>
            <li><a href="#">Subitem 1</a></li>
            <li><a href="#">Subitem 2</a></li>
            <li><a href="#">Subitem 3</a>
                <ul>
                    <li><a href="#">Subitem 3-1</a></li>
                    <li><a href="#">Subitem 3-2</a></li>
                    <li><a href="#" class="active">Subitem 3-3</a></li>
                </ul>
            </li>
        </ul>
    </li>
    <li><a href="#">Category 2</a>
        <ul>
            <li><a href="#">Subitem 1</a></li>
            <li><a href="#">Subitem 2</a></li>
        </ul>
    </li>
</ul>

Usage:

$(document).ready(function(){
	$('#vmenu').vmenu();
});

You can use it for AJAX requests:

$(document).ready(function(){
	vMenu.forward=function(elem) {
		// your action here
		return false;
	}
});
